# Introduction-to-Robotics-DD2410-Assignments-Project-Work-with-ROS

This repository contains assignments and project work from the course **DD2410 – Introduction to Robotics** at KTH Royal Institute of Technology. The course builds a strong foundation in robotics by combining theory with hands-on implementation. It covers robot kinematics, motion planning, control, perception, localization, mapping, and autonomous navigation, along with creative extensions beyond the original course requirements.

---

## Development Environment

- **Operating System:** Ubuntu 20.04 Arm64 (via UTM VM on MacBook M1 Pro)
- **Middleware:** ROS Noetic
- **Programming Language:** Python

---

## Repository Structure

### `catkin_ws`
ROS workspace containing four laboratory assignments:

- **Assignment 1** – ROS Basics  
- **Assignments 2–4** – Kinematics, Planning, and Mapping  

### `project_ws`
ROS workspace containing one major robotics project:

- **Project Assignment** – Tiago Mobile Pick-and-Place  

---

## Purpose

This repository not only documents coursework but also includes additional self-developed features where assignments were extended with creative improvements, alternative implementations, and experimental ideas to explore robotics concepts more deeply for my own learning.
